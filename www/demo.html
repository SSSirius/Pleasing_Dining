Pleasing Dining

Pleasing Dining is a mobile app, which cooperates with school canteens and provides a booking system and nutrition plan for students.
The app consists of four main Tabs:
Home
Search
Information
Recommendations
Setting Goals
Daily Data
Sports
Sports Records
Daily Consumption Data
Entire Source Code
Meals
Selecting Canteen
Booking Dishes
Daily Evaluation of Nutrition
Meals Records
Dishes Introductions
Me
Personal Profile
Personal Data
Diet Status
Advice
Setting Goals
